# 🕌 Zakat Calculator

A simple and user-friendly **Zakat Calculator** built to help Muslims calculate their annual Zakat accurately according to Islamic principles.

## ✨ Features
- Calculate Zakat on:
  - Gold & Silver
  - Cash Savings
  - Business Assets
  - Property & Investments
  - Income and Other Assets
- Automatic Nisab eligibility check
- Accurate annual Zakat calculation (2.5%)
- Clean and easy-to-use interface

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

## 📌 Purpose
This project aims to simplify the process of calculating Zakat by providing an accessible digital solution for Muslims worldwide.

## 🚀 How to Use
1. Enter your assets details (gold, cash, business assets, etc.)
2. Check if your wealth meets the Nisab threshold
3. Get your total payable Zakat instantly
